# Audio Prompts — Suno AI v2 (Estilo Retro Gaming)

## Instrucoes
1. Acesse suno.com
2. Cole o prompt no campo "Song Description"
3. **MARQUE "Instrumental"** (sem vocal!)
4. Duracao: 1-2 minutos (vai fazer loop)
5. Gere 2 versoes de cada e escolha a melhor
6. Baixe .mp3, renomeie conforme indicado
7. Coloque na pasta Downloads

## Dicas pro Suno
- Se a musica tiver vocal, regenere com "Instrumental" marcado
- Se ficar muito "moderno", adicione "NES, SNES, Game Boy, retro console" ao prompt
- Se ficar longo demais, ta ok — o jogo faz loop automatico
- Priorize: **cemetery, boss, menu, tokyo** (sao os mais importantes)
