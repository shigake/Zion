# Audio Prompts — Suno AI

## Instrucoes
1. Acesse suno.com e crie conta (free tier = 10 musicas/dia)
2. Cole cada prompt abaixo no campo "Song Description"
3. Marque "Instrumental" (sem vocal)
4. Duracao: 2-3 minutos cada (vai fazer loop no jogo)
5. Baixe o .mp3 e renomeie conforme indicado
6. Coloque todos na pasta Downloads
