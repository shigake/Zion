# Zion

Jogo estilo Vampire Survivors / The Spell Brigade.

## Sobre

Zion e um survivors game com tematicas variadas, personagens unicos e sistema de progressao entre runs. O objetivo e sobreviver a hordas de inimigos cada vez maiores enquanto evolui armas e coleta itens.
